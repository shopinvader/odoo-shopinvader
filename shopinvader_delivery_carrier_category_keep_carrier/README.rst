This module allows to avoid updating the carrier when the shipping adress
is updated.

This is usually useful when a carrier allows to deliver on third party
(fuel station, ...).

**Table of contents**

.. contents::
   :local:

Installation
============

This module needs 'delivery_carrier_cateogory'.

Usage
=====

Go to Delivery carriers category and check the box 'Keep Carrier On Shipping
Change'.
