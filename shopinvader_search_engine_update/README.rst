This module extends the functionality of the
``shopinvader_search_engine`` to mark the information exported to the
search engine as to be recomputed when the product data are updated.

.. IMPORTANT::
   This is an alpha version, the data model and design can change at any time without warning.
   Only for development or testing purpose, do not use in production.
   `More details on development status <https://odoo-community.org/page/development-status>`_

Use Cases / Context
===================

When you update your product data and this information is exposed by the
search engine, you expect to see your changes applied into the search
engine as soon as possible.
